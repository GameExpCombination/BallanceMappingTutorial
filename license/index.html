Copyright and License¶

At the beginning of establishing this tutorial, for decreasing the useless work, I select the 2 books which is published by GCBA, "3ds max 在 Ballance 制图中的应用" and "Virtools制图 第二版". And I construct this tutorial under the basis of these 2 books. Before I create this tutorial, due to that these 2 books don't use any public license, I send the application about authorization to these books' main maintainers via QQ, a Chinese instant-messaging software. And after I get the agreement of them, I apply a new public license, CC BY-NC-SA 4.0, to these 2 books and use them legally to build this website.

The referenced books about these 2 books, are seen as that all of them has been put in public territory. And I only list their authors' name.

Due to the error of construction, this project's first commit is under CC BY-SA 4.0. Following commit has corrected this error. So, don't use the first commit of this project because it is illegal.
